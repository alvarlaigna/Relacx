Installation

For using Relacx you need to include 4 resources,

React JS
React-Dom JS
Babel Helpers
Relacx JS

Since Relacx is a state manager for React, it requires React.js and React-Dom.js to be available
The current tested version of Relacx uses React version 15.5.4

The resources available in the script tag below can be used or you can also visit React's official page and follow your choice of installation for both React and React-Dom. 

Non-Minified
<script type="text/javascript" src="https://webxtended.github.io/Relacx/resources/react.js"></script> <script type="text/javascript" src="https://webxtended.github.io/Relacx/resources/react-dom.js"></script> 
Minified
<script type="text/javascript" src="https://webxtended.github.io/Relacx/resources/react.min.js"></script> <script type="text/javascript" src="https://webxtended.github.io/Relacx/resources/react-dom.min.js"></script>

Then you need the babel helpers JS if you are transpiling your code using babel's ES6 converter

<script type="text/javascript" src="https://webxtended.github.io/Relacx/resources/helpers.js"></script> <script type="text/javascript" src="https://webxtended.github.io/Relacx/resources/helpers.min.js"></script>

Finally include Relacx JS

<script type="text/javascript" src="https://webxtended.github.io/Relacx/resources/relacx.js"></script> <script type="text/javascript" src="https://webxtended.github.io/Relacx/resources/relacx.min.js"></script>
